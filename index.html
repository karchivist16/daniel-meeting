안건 1: 4주차 3주차 새가족 교육 때 간사 방문

배경/문제

  • 새가족 명단이 편입 2주 전(4주차)에 넘어오고, 5주차에 셀 편입이 이루어짐
  • 서면 정보(루디아 기록)만으로는 새가족의 실제 분위기·성향을 파악하기 어려움
  • 기존에는 간사가 새가족을 편입 전에 직접 만나는 절차가 없었음

논의 요약

  • 4주차에 해당 청 간사들이 루디아로 가서 새가족을 직접 만나자는 제안
  • 얼굴 익히고, 5~10분 대화하면서 어떤 셀이 맞을지 파악하는 목적
  • 간사가 가는 것 vs 새가족이 오는 것: 새가족 입장에서는 익숙한 환경(루디아)에서 만나는 게 더 자연스러움
  • 변수: 새가족 출석여부를 새가족부장도 당일 돼봐야 아는 경우가 있어 항상 대기해야 하는 상황이 될 수 있음
  • 체계를 마련하는 것 자체에는 전원 찬성. 일단 시행 후 수정·보완하기로

결정 사항

  • 시행: 4주차 새가족 교육 때 해당 청 간사들이 루디아 방문
  • 예외 상황이 자주 생기면 방식 수정·보완
  • 루디아 측에 4주차 일정 공유 요청 (1청 / 2청은 현재 시행 중)
📌 업데이트 (5/3): 4주차 → 3주차로 변경. 간사가 3주차 새가족 교육 때 방문하는 것으로 확정.

안건 2: 편입 후 1개월 간사 체크

배경/문제

  • 루디아에서 편입 후 1개월간 새가족에게 직접 연락하고 있음
  • 그러나 셀 내부 적응 상태는 루디아가 파악하기 어려움
  • 실제 사례: 편입된 새가족이 셀 분위기에 적응하지 못하고 셀모임 불참 → 예배만 듣고 나감. 이 사실을 아무에게도 말하지 않아서 모르고 있다가 루디아에서 우연히 알게 됨

논의 요약

  • 루디아가 새가족 본인에게 연락하니, 간사들은 셀장을 통해 확인하는 역할 분담
  • 단순 출석 여부가 아니라 나눔 참여도·태도·적응 상태까지 파악 필요
  • 기간(1개월) vs 횟수(4회) 논의: 띄엄띄엄 나오면 한 달로는 파악 어려움 → 4회 참석 기준이 관리에도 용이
  • 확인 방법은 간사 자율 (셀에 직접 들어가 보기, 셀장에게 연락 등)

결정 사항

  • 4회 참석 기준으로 새가족 적응 체크
  • 간사가 셀장에게 확인 (출석 + 적응 상태)
  • 확인 방법은 각 간사 자율

안건 3: 유예기간 도입 → 편입 후 케어 프로세스

배경/문제

  • 편입 후 셀에 적응 못하거나 셀 변경을 원하는 경우가 종종 있음
  • 나이가 있는 청년부 특성상 맞고 안 맞고가 더 확실함
  • 현재는 셀 변경에 대한 공식 절차/매뉴얼이 없음

논의 요약

  • 편입 후 1개월 정도(4회 출석)를 유예기간으로 설정하고, 마지막 주에 셀 적응 확인 → 변경 원하면 2주 검토 후 재배정하는 구조
  • 셀 변경 의사 확인 주체: 간사가 물어봐도 "괜찮아요" 인사치레할 가능성 → 루디아(새가족부장)가 마지막 연락 시 확인하는 게 더 자연스러움
  • 간사들은 셀장 쪽에 포커싱
  • 셀 변경은 최대한 안 일어나는 게 목표 (새가족 교육 4주차 방문도 그 목적). 그래도 생기면 대응할 매뉴얼이 필요
  • 변경 요청 시 2주 검토 기간이면 셀장들에게도 부담 적을 것

결정 사항 (4/26 원안)

  • 유예기간 1개월 도입
  • 셀 변경 의사 확인: 루디아가 마지막 연락 시 담당
  • 변경 요청 시: 2주 검토 기간 → 간사 내부 논의 → 셀 제안/배정
  • 목사님 동의 필요 → 다음주에 보고
📌 업데이트 (5/3, 목사님 의견): 유예기간 도입 대신, 안건 4(심방)와 통합하여 편입 후 케어 5단계 프로세스로 대체.

안건 4: 새가족 편입 후 심방 → 안건 3에 통합 (5/3)

배경/문제

  • 셀장이 새가족 심방을 아예 안 하는 경우가 꽤 있음
  • 새가족 입장: 루디아에서는 집중 케어를 받다가, 셀 편입 후 관심을 덜 받는 느낌
  • 심방 미실시 → 새가족이 소외감 → 셀 이탈로 이어질 가능성이 높음

논의 요약

  • 유예기간 1개월 내 심방 무조건 필수. 바쁘면 전화라도, 예배 전 카페에서 20분이라도
  • 간사가 셀장에게 "심방했는지" 지속적으로 확인하는 역할
  • 심방 미실시 셀장은 다음 새가족 배정 시 후순위로 → 목사님께 말씀드려 공식화하자는 의견
  • 간사가 직접 새가족 심방을 하면 어떨까? → 루디아에서도 궁극적으로 그걸 원함. 다만 셀장 심방이 기본이고, 그것조차 안 되는 게 현재 문제. 일단 셀장 심방부터 정착시키기로

결정 사항 (4/26 원안)

  • 새가족 심방 필수 (유예기간 내)
  • 간사가 셀장에게 심방 여부 지속적으로 확인
  • 심방 미실시 셀장 → 다음 새가족 배정 후순위 (목사님 동의 필요)
📌 업데이트 (5/3, 목사님 의견): 안건 3의 편입 후 케어 5단계 프로세스에 통합. 심방 시점이 "1개월 내" → "2주 이내"로 구체화되고, 간사 확인 + 사역자 심방 연결이 추가됨.

안건 5: 주일 교육 출석

배경/문제

  • 토요일 교육: 직장인 사정 이해하여 목사님도 양보
  • 그 대신 주일 교육을 실시하고 있는데, 주일도 안 오는 셀장이 많음
  • 교육 안 듣는 셀장 → 셀교육 통한 나눔 없이 셀모임 진행 → 셀원들 사이 불만 발생

논의 요약

  • 목사님이 몇 번이나 주일 교육 참석을 말씀하셨으나 안 지켜지는 상황 → 간사들이 직접 체크해야
  • 당분간 주일 교육 출석 체크 → 간사방에 공유하기로

결정 사항

  • 주일 교육 출석 체크 시작 → 간사방에 공유
  • 간사들이 셀장들에게 주일 교육 참석 독려
  • 하원셀의 경우 토요일·주일 모두 어려운 상황 → 목사님과 별도 논의 필요

결정 사항

  • 담당 셀장과 간사가 연락을 지속하기로
